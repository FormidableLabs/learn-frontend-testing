talk, "[Learn Frontend Testing][meetup]".

[seattlejs]: http://www.meetup.com/seattlejs/
[meetup]: http://www.meetup.com/seattlejs/events/139993642/


Building the Slide Deck
=======================

To re-build this slide deck from Jade sources:

    $ npm install
    $ grunt jade

To watch changes:

    $ grunt watch
